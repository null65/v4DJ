I recently deployed a dedicated MeshCore Repeater node (MeshCore.co.uk) to create a resilient, off-grid text messaging network for local enthusiasts. This decentralized, multi-hop routing platform runs on a Heltec V3 LoRa board.

The Heltec V3 repeater with the 6dB omni antenna, mounted securely in the loft.

Installation for Off-Grid LoRa Routing

The repeater node is placed high up in the loft, utilizing a 6dB gain omnidirectional antenna to ensure maximum reach and coverage for message routing. I use another Heltec V3 as a companion client node which allows me to reliably send and receive text messages on the mesh network from my shack and while portable. The setup is simple but extremely effective, and provides robust, peer-to-peer communication.

Multi-Hop Routing Performance and Paths

I'm receiving great performance reports from other local users regarding message delivery. My repeater node intelligently handles multi-hop routing to extend the reach of the mesh. It reliably routes text packets through the local DDRC omni and provides a solid path to the repeater located at the VARC radio cabin, significantly expanding the network's accessible range for everyone nearby. This project has been a huge success for improving resilient, off-grid communications.
